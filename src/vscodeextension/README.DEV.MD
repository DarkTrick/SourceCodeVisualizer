
# Development

## build
- build *web_visualizer*
  - (from here) `npm run build_dependencies`
- copy *web_visualizer* release to here
  - `npm run get_dependencies`
- `npm run compile`

## run

- Preparation: get latest web visualizer
  - build *web_visualizer*
    - (from here) `npm run build_dependencies`
  - copy *web_visualizer* release to here
    - `npm run get_dependencies`
- open `vscodeextension`
- Menu -> Run -> `Run & Debug`
